<예습> - 시간초과 실패

```python
n = int(input())
count = 0
for i in range(2, n+1):
    for j in range(2, int(i**0.5)+1):
       if i % j == 0:
            break
    else:
        count += 1
print(count)
```

<강의>

```python
n = int(input())
ch = [0]*(n+1)
cnt = 0
for i in range(2, n+1):
		if ch[i] == 0:
				cnt += 1
				for j in range(i, n+1, i):
						ch[j] = 1
print(cnt)
```

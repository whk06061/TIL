<내 코드>

```python
import sys
#sys.stdin = open("input.txt", "r")
n = int(input())
Player = []
for _ in range(n):
    h, w = map(int, input().split())
    Player.append((h, w))
Player.sort(reverse=True)
tallest = 0
heaviest = 0
cnt = 0
for h, w in Player:
    if h >= tallest or w >= heaviest:
        cnt += 1
        if h >= tallest:
            tallest = h
        if w >= heaviest:
            heaviest = w
print(cnt)
```

<내 코드>

```python
import sys
#sys.stdin = open("input.txt", "r")
L = int(input())
Box = list(map(int, input().split()))
M = int(input())
for i in range(M):
    Box[Box.index(max(Box))] -= 1
    Box[Box.index(min(Box))] += 1
print(max(Box) - min(Box))
```

import sys
#sys.stdin = open("input.txt", "r")
L = int(input())
a = list(map(int, input().split()))
m = int(input())
a.sort()
for _ in range(m):
    a[0] += 1
    a[L-1] -= 1
    a.sort()
print(a[L-1]-a[0])

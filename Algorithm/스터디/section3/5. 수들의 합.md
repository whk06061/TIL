<내 코드> - 시간초과 실패

```python
N, M = map(int, input().split())
a = list(map(int, input().split()))
cnt = 0
for i in range(N):
    sum = 0
    for j in range(i,N):
        sum += a[j]
        if sum >= M:
            if sum == M:
                cnt += 1
            break
print(cnt)
```

<강의 코드>

```python
n, m = map(int, input().split())
a = list(map(int, input().split()))
lt = 0
rt = 1
tot = a[0]
cnt = 0
while True:
    if tot < m:
        if rt < n:
            tot += a[rt]
            rt += 1
        else:
            break
    elif tot == m:
        cnt += 1
        tot -= a[lt]
        lt += 1
    else:
        tot -= a[lt]
        lt += 1
print(cnt)
```

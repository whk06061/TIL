<내 코드>

```python
a = []
for i in range(0, 21):
    a.append(i)
for i in range(10):
    start, end = map(int, input().split())
    for j in range(start, (start+end)//2+1):
        tmp = a[j]
        a[j] = a[end+start-j]
        a[end+start-j] = tmp
for i in range(1, 21):
    print(a[i], end=" ")
```

<내 코드2>

```python
a = []
for i in range(0, 21):
    a.append(i)
for i in range(10):
    start, end = map(int, input().split())
    tmp = a[end:start-1:-1]
    idx = 0
    for j in range(start, end+1):
        a[j] = tmp[idx]
        idx += 1
for i in range(1, 21):
    print(a[i], end=" ")
```

<강의 코드>

```python
# 0부터 20까지 리스트를 만들어줌
a = list(range(21))
for _ in range(10):
    s, e = map(int, input().split())
    for i in range((e-s+1)//2):
        a[s+i], a[e-i] = a[e-i], a[s+i]
a.pop(0)
for x in a:
    print(x, end=' ')
```

파이썬에서 두 수 스왑하는 방법.
a, b = b, a 이렇게 간단하게 스왑할 수 있다.

직접 힙 구조로 구현해보기

<강의 코드>

```python
import sys
import heapq as hq
#sys.stdin = open("input.txt", "r")
a=[]
while True:
    n = int(input())
    if n == -1:
        break
    if n == 0:
        if len(a) == 0:
            print(-1)
        else:
            print(hq.heappop(a)) #루트 노드 값 pop 시켜줌 그리고 최소힙 형태 만들어줌
    else:
        hq.heappush(a, n) #최소힙의 형태로 넣음
```

<예습>

```python
import sys
t = int(input())
for i in range(t):
    n, s, e, k = map(int, input().split())
    input_num = list(map(int, sys.stdin.readline().split()))
    input_num = input_num[s-1:e]
    input_num.sort()
    print(f'#{i+1} {input_num[k-1]}')
```

```python
import sys
t = int(input())
for i in range(t):
    n, s, e, k = map(int, input().split())
    input_num = list(map(int, sys.stdin.readline().split()))
    input_num = input_num[s-1:e]
    input_num.sort()
    print("#%d %d" %(i+1, input_num[k-1])
```

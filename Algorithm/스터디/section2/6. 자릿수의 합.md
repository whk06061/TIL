<예습>

```python
# 함수 정의
def digit_sum(x):
    sum = 0
    for i in x:
        sum += int(i)
    return sum

# 입력받음
input()
input_nums = input().split()

# {입력값 : 각자리수}의 합 형식으로 딕셔너리 저장 예정
sum_dict = {}

# 입력받은 값의 각자리 수 합 구해서 딕셔너리에 저장
for value in input_nums:
    sum_dict[value] = digit_sum(value)

# 가장 큰 자리수 저장
max_value = max(sum_dict.values())

# 그 값을 갖는 최초의 키 출력
for key, value in sum_dict.items():
    if value == max_value:
        print(key)
        break
```

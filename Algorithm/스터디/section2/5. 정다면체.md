<예습>

```python
n, m = map(int, input().split())
sum_dict = {}
for i in range(1, n+1):
    for j in range(1, m+1):
        if sum_dict.get(i+j) == None:
            sum_dict[i+j] = 1
        else:
            sum_dict[i+j] = sum_dict[i+j]+1
max_value = max(sum_dict.values())
for key, value in sum_dict.items():
    if value == max_value:
        print(key, end=" ")
```
